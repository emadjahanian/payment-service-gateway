# Payment Gateway Server

A demo payment gateway server providing both **REST** and **SOAP** APIs.  
Built with **Spring Boot**, **PostgreSQL**, and **JPA** to simulate payment processing and store transactions.

## Features
- REST endpoints for integrations
- SOAP endpoints for legacy systems
- PostgreSQL database with JPA entities
- Repository layer for persistence
- Docker support (optional)

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

## Getting Started
```bash
mvn clean install
mvn spring-boot:run
